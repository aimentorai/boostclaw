# development-sheet-factory 使用说明

## 触发指令

> 生成开发表格

> 帮我生成 20260329 的开发表格

> 批量生成开发表格，复筛表在 20260329选品/20260329选品复筛.xlsx

## 前置条件

需要先有 `YYYYMMDD选品复筛.xlsx`（由 product_chart_get 生成）。

## 输出文件

输出目录：`YYYYMMDD选品/YYYYMMDD开发表格/`

| 文件 | 说明 |
|------|------|
| `{SPU_ID}+{中文标题}.xlsx` | 各产品开发表格（含主图） |
| `_READY.json` | 生成完成记录 |

## 表格字段说明

| 单元格 | 内容 |
|--------|------|
| C2 | 亚马逊商品链接 |
| F2 | 主图（嵌入图片，下载失败时写 URL） |
| O2 / P2 | 季节性产品 / 节日月份 |
| B7 | 中文标题 |
| I7 / J7 | 售价 |
| J2 / K2 / N2 | 1688供应商链接 / 采购价 / 材质（由询盘回填） |
| D7 / F7 / G7 / H7 | 重量 / 长 / 宽 / 高（由询盘回填） |

## 典型用法

```
# 单独使用
生成开发表格，复筛表：20260329选品/20260329选品复筛.xlsx

# 作为完整开品流程的一部分（自动调用）
帮我做 20260329 选品的完整开品流程
```
