# product-chart-get 使用说明

## 触发指令

直接说：

> 抓取天猫选品数据

> 帮我抓第1页，ds=20260328

> 默认（第1页，ds=昨天）

## 参数说明

| 参数 | 默认值 | 说明 |
|------|--------|------|
| 页码 | 第1页 | 每页50条 |
| ds | 昨天日期 | 数据分区日期，格式 YYYYMMDD |

## 输出文件

输出目录：`YYYYMMDD选品/`

| 文件 | 说明 |
|------|------|
| `data-get-第x页数据.json` | 原始抓取数据 |
| `ai_results.json` | AI 预处理结果（中文标题/复筛/季节性） |
| `YYYYMMDD选品.xlsx` | 全量选品表 |
| `YYYYMMDD选品复筛.xlsx` | 复筛通过表（下游步骤的输入） |
| `已排查产品.xlsx` | 历史剔除产品追加记录（位于选品目录同级） |

## 典型用法

```
# 单独使用
抓取天猫选品数据，第1页，ds=20260328

# 作为完整开品流程的第一步
帮我做 20260329 选品的完整开品流程
```
